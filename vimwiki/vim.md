# vim

## Object (Nouns)
* w - word
* W - WORD
* s - sentence
* p - paragraph
* t - tag
 
## Modifiers
i  - inner
a  - a as

## Register actions
" - select register to use for action

## Targeting

* w - first character in next word
* W - first character in next WORD
* e - last character in next word
* E - last character in next WORD
* b - previous word's first character or first character of this word

## Navigation
* `cTRL-]` - jump to
* `CTRL-0` - jump back
